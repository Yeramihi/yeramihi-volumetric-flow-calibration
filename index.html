Yeramihi Volumetric Flow Calibration

Description

This tool started with a simple question: how fast can this filament actually print on my printer?

My first 3D printer was an Ender-3 V3 SE. I bought it with two spools of Ender filament, rated for 30-55mm/s. Later, when I bought a Bambu P2S, I wanted to run a volumetric speed test and see what that same filament could really do on a faster machine.

The problem was that the built-in max-flow calibration model was wasteful for what I wanted to learn. The default 5-20mm³/s range used about 30 grams of filament, and it still had to be sliced through the slicer using an existing filament profile. That meant the result was not just a filament test. It was a filament test filtered through slicer settings, profile limits and assumptions that I was specifically trying to measure past.

So I made this browser tool to generate raw G-code directly. No model to slice, no hidden slicer limits, just a repeatable stepped-speed pattern based on the printer, bed plate, nozzle size, temperature, cooling and filament being tested. The goal is to find the practical maximum volumetric flow for a real setup instead of relying only on marketing numbers or default profile values.

On my P2S plate, the G-code produced by this tool can test the same 5-20mm³/s range using a little over 1 gram of filament, including the printed flow numbers, instead of about 30 grams.

One thing this test is very sensitive to is first-layer adhesion. The bed plate needs to be properly clean before running it, because even a tiny adhesion problem can lift or distort one of the long calibration lines. When that happens, the result is no longer a useful flow test; it is just a failed first layer pretending to be flow data.

The result that started all of this surprised me: the black Ender PLA I had was able to print at a sustained maximum flow rate of 19mm³/s on my setup, compared with the 5mm³/s value defined by the manufacturer. That gap is exactly why this tool exists.

Safe Y is used for the prime/calibration line, so it must be an actual printable Y position that the nozzle can reach.

Disclaimer & Caveats

This tool generates a simple volumetric-flow speed calibration G-code file. It helps identify the practical flow limit of a filament, nozzle and printer setup.

This tool is intended for experienced users who understand G-code, printer movement, extrusion behaviour, and machine limits.

Assumptions

  • The filament is already loaded and ready to extrude.
  • Bed mesh / bed levelling has already been completed.
  • Your normal Z offset is already calibrated.
  • The optional Z offset adjustment is only for small additional correction.
  • The safe starting position represents the minimum reachable safe XY position.
  • The generated pattern applies a 10mm margin from the safe starting position.

Z Offset Note

Some printer and plate combinations may apply additional Z behaviour in their normal start G-code. For example, a textured plate may require a small additional Z-plane lift, such as around 0.1mm, depending on the machine profile.
